## 🚀 快速开始

### 5 分钟快速上手

1. **查看设计系统预览**
   ```bash
   npm run dev
   # 访问 http://localhost:5175/design-system
   ```

2. **阅读快速参考指南**
   - 打开 [quick-reference-guide.md](./quick-reference-guide.md)
   - 浏览颜色、字体、间距速查表

3. **尝试第一个迁移**
   ```jsx
   // Before
   <div className="text-ds-cod">Hello</div>

   // After
   import { colors } from '../design-system/tokens';
   <div style={{ color: colors.grey[9] }}>Hello</div>
   ```

4. **遇到问题？**
   - 查看 [设计系统迁移映射文档](./design-system-migration-mapping.md)
   - 找到对应的组件或样式映射
